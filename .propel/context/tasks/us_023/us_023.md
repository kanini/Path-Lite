# User Story - US_023

## Story ID
* ID: US_023

## Story Title
* Mandatory Field Enforcement

## Description
* As a nurse, I want mandatory fields enforced during data entry with clear indicators, so that I cannot skip required fields and the form cannot be submitted incomplete.

## Acceptance Criteria
1. **Given** I view the form, **When** the screen loads, **Then** the system displays asterisk (*) indicators on all 9 mandatory fields: MRN, First Name, Last Name, DOB, Gender, Treatment Location, Room Number, HBsAg, HBsAb.

2. **Given** I try to skip a mandatory field, **When** AI asks a required question, **Then** the system prevents skipping by re-asking the question until valid input is provided.

3. **Given** I enter invalid data, **When** validation fails on a mandatory field, **Then** the system announces error via TTS (e.g., "MRN must be numeric. Please provide a valid Medical Record Number") and re-asks the question.

4. **Given** optional fields are encountered, **When** AI asks an optional question, **Then** the system allows skipping by accepting "skip" or "none" responses and stores NULL value.

5. **Given** I attempt submission, **When** any mandatory field is incomplete, **Then** the system blocks submission, displays error message "Please complete all required fields", and highlights incomplete fields.

## Edge Cases
* What happens when nurse says "I don't know" for mandatory field? (Treat as invalid response; re-ask question with clarification "This field is required. Please provide the [field name].")
* How does the system handle partial mandatory field completion? (Track completion status per field; allow session save but prevent final submission until all mandatory fields complete)
